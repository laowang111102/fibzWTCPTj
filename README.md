# 学生就业管理系统 python345

学生就业管理系统是一个基于 Django 框架开发的平台，主要用于管理学生的就业信息，涉及学院、专业、企业、岗位等多方面数据的维护。

## 技术栈

- 后端：Python 3, Django
- 前端：HTML, CSS, JavaScript
- 数据库：MySQL

## 功能模块

### 系统角色

#### 管理员功能

- 系统首页
- 学院信息管理
- 专业信息管理
- 企业信息管理
- 岗位信息管理
- 管理员信息管理
- 学生信息管理
- 学生教育经历管理
- 学生项目经历管理
- 学生投递记录管理

#### 招聘方功能

- 系统首页
- 岗位信息管理
- 学生信息管理
- 学生教育经历查看
- 学生项目经历查看
- 学生投递记录查看

#### 学生用户功能

- 系统首页
- 岗位信息浏览
- 教育经历设置
- 项目经历设置
- 投递记录查看

## 系统运行环境

- Python 3.8 或以上版本
- Django 3.x 或以上版本
- MySQL 5.7 或以上版本

## 部署步骤

1. 安装 Python 和 Django
2. 安装 MySQL 数据库，并创建数据库及用户
3. 复制项目代码到服务器
4. 执行数据库迁移命令，以应用模型更改
5. 配置前端静态文件
6. 运行项目

## 目录结构

```
.
├── apps/                # 应用目录
│   ├── admin/           # 管理员界面相关
│   ├── enterprise/      # 企业信息相关
│   ├── student/         # 学生信息相关
│   └── ...
├── static/              # 静态文件目录
├── templates/           # HTML模板目录
├── manage.py            # Django管理脚本
└── ...
```

- 使用 Django 框架，确保系统的稳定性和可维护性
- 灵活的数据库设计，支持扩展和定制开发
